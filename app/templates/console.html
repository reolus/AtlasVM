Console requirements

What AtlasVM expects

The VM must be running and libvirt must expose an active VNC display, usually from a graphics device like <graphics type='vnc' autoport='yes' listen='127.0.0.1'>.

What this page will not do

It will not undefine/redefine the VM to add graphics. That keeps the console fix aligned with the snapshot-safe rule instead of detonating configuration history for convenience.
